Ce ci est mon projet de BTS CIel en collaboration avec [https://github.com/Elian-BROLY](url) et [https://github.com/LeCyril05](url).

Notre projet à pour but de réaliser une station méteo dons les données seront observable via un site web et un application android

Dans le cadre du projet, je suis chargé de : 
  - Définir le modèle de base de la base de donnée
  - Mettre en oeuvre un serveur pour une BDD (MySQL) et l'hébergement d'un site WEB
  - Développer un site WEB pour affichage ergonomique des valeurs d'un site WEB
  - Sécuriser l'accès à ce site par une identification
    - Création de compte utilisateurs avec mot de passe
    - Enregistrement des utiliateurs dans une BDD
    - Créer un compte administrateur et une page pour gérer les utilsateurs
    - Envoyer un mail à l'administrateurs en cas de trois échecs d'identification

Les critères de réussite sont :
- La BDD et l'hébergement du site web sont opérationels
- Le site permet de visulaiser les données. L'affichage est conviviale et facile à utiliser.
- L'accès aux informations est sécurisé et résiste à une attaque (injetcion, force brute...)
